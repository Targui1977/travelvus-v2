The Telegraph 2025 ranking (Gatwick 6th, Stansted 29th) measures cancellation rates, security times, and punctuality. This is a single newspaper study, not an official CAA metric. It cannot anchor a core decision mechanic without official verification. As editorial context: useful. As a core page mechanic: not defensible.

**Corrected capability label:** Drop "QUALITY-VS-PRICE." Use **ACCESS FRICTION-VS-PRICE** — the extra train time and cost to reach Stansted vs Gatwick, measured with objective transport data.

## G. STRONGEST BOUNDARY

### **ACCESS-FRICTION THRESHOLD**

Stansted's cheaper flight must save enough to compensate for the longer, more expensive train journey.

**Equation:**
```
STN_advantage = STN transfer penalty - flight saving
STN wins when flight_saving > transfer_penalty
```

**Canonical example (Victoria, 1 adult):**
- Flight saving: €95 − €75 = €20
- Transfer penalty: €23.17 − €12.52 = €10.65
- STN still ahead by: €20 − €10.65 = €9.35 → **rounded to €9**

**"The Line":** At what flight-price gap does Stansted's transfer penalty erase the saving? When flight saving = transfer penalty = ~€11. If STN is only €11 cheaper, it's a tie. Below €11, Gatwick wins.
