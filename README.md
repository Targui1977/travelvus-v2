# Travelvus V2

A second opinion before you book. Compare two flights and find out which really wins.

## Dev

```bash
npm install
npm run dev      # http://localhost:3000
npm test         # 27 tests
npm run build    # production build
```

## Routes

- `/` — Quick Compare
- `/result` — Full Result + Verdict Changed
- `/compare/heathrow-vs-stansted` — Decision Page

## Stack

Next.js · TypeScript · Tailwind CSS · Vitest
